// turbo-all
Follow these steps to deploy your changes and the built version to GitHub.

1. **Build the Project**:
   ```bash
   npm run build
   ```

2. **Commit Changes & Build Output**:
   ```bash
   git add .
   git commit -m "build: update project and build artifacts"
   ```

3. **Push to GitHub**:
   ```bash
   git push
   ```
