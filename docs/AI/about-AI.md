# 关于AI的一些个人思考
曾经的巨型电脑,如今的手提,平板,手机;
曾经的固话拨号上网,如今的FTTR,MESH千兆宽带;
曾经的32M内存,如今的1T,2T硬盘;

曾经的手动翻译upload,如今的Agent,Antigravity,Gemini In Chrome,直接翻译,直接上传.(是的,此时此刻的我就是在用Antigravity完成这个网站,上传新的文章).  

我不知道这种日记类,纯个人想法的篇章还会有多少.因为我能想到的关于这个网站之前的一切,都可以使用AI来完成.  

所幸,我对AI并不抵触,觉得它非常有趣,非常全能:"我想做什么?我自己能不能完成?如果不能,AI能不能替我完成?如果能就太好了;如果不能,我能不能通过学习AI,为AI赋能,让它学着做呢?"  
于是,我想借这篇文章,作为自己翻页新篇章的开始.  
我不知道我对AI的热情会是3分还是10分.但我知道,我对AI是感兴趣的.  
我也不知道自己感兴趣的话题,会一路指引到我去哪.今天是Prompt Engineering, 明天会不会是LLM的构建?后天会不会是MCP?还是Agent Skills?  
虽然可能不能成为制造工具的人,但工具诞生了,会用了,总不会被时间浪潮所淹没.  
像是如今老人不会使用智能手机,作为新时代"老人",不会使用AI会不会就是不同时间的相同处境?
